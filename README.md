# RUST-LEARNING

## 主要コマンド

```
cargo fmt // フォーマット修正
cargo clippy // 静的解析
cargo add <ライブラリ名>
cargo check // コードチェック
cargo build //
cargo run //
cargo clean //
cargo test //
cargo help //
```
